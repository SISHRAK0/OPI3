git index - промежуточная область, где хранятся подготовленные файлы перед коммитом
просмотр - git ls-files --stage

git hooks - грубо говоря команды, которые выполняются перед какими-то действиями, например перед коммитом, можно залезть под капот и написать свое, чтобы допустим тесты запускались и тд
просмотр - ls -1 .git/hooks

git хранит все как снимки, снапшоты в виде объектов
blob - содержимое файлов
tree - структуры директорий, ссылаются на blob’ы и другие tree’ы
Commit‑объект — указывает на один tree (состояние проекта), содержит метаданные (автор, дата, сообщение) и ссылки на родительские коммиты (SHA‑1 родителей)

просмотр всех коммитов - git log --graph --oneline --decorate --all
